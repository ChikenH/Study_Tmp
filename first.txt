Git の練習をしていきます。
基本的なワークフローを学びます。
* git add
* git commit
* git rm
* git status
* git diff

---

# ワークフロー

## 初期化
* ローカルリポジトリを作成
> git init
* 確認
> ls -a

## 作業
* 変更差分を確認(ステージングエリアとの差分)
> git diff
* 変更差分を確認(ステージングとコミットとの差分)
> git diff HEAD

* ステージングエリアに追加(存在するファイル)
> git add [ファイル名]
> git add .

* ステージングエリアに追加(削除したファイル)
> git rm [ファイル名]

* ローカルリポジトリの状態(コミットからの差分)を確認
> git status

## コミット(ローカル)
* ローカルリポジトリにコミット
> git commit -m [コミットメッセージ]
* 差分を確認してからコミット
> git commit -v

* 履歴を確認
> git log -[遡って参照するログ数]
* 履歴を一行で表示
> git log --oneline
* ファイルの差分を表示
> git log -p [ファイル名]
* Vim から出る
> :q

## コミット(リモート)
* リモートリポジトリにコミット
> git push -u origin master
