# ワークフロー

## 初期化
* ローカルリポジトリを作成
> git init
* 確認
> ls -a

## 作業
* ステージングエリアに追加
> git add [ファイル名]
> git add .

* ローカルリポジトリの状態を確認
> git status

## コミット(ローカル)
* ローカルリポジトリにコミット
> git commit -m [コミットメッセージ]

## コミット(リモート)
* リモートリポジトリにコミット
> git push -u origin master
