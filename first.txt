基本的なワークフローを学びます。
* git add
* git commit
* git rm

---

# ワークフロー

## 初期化
* ローカルリポジトリを作成
> git init
* 確認
> ls -a

## 作業
* ステージングエリアに追加(存在するファイル)
> git add [ファイル名]
> git add .

* ステージングエリアに追加(削除したファイル)
> git rm [ファイル名]

* ローカルリポジトリの状態を確認
> git status

* 履歴を確認
> git log

## コミット(ローカル)
* ローカルリポジトリにコミット
> git commit -m [コミットメッセージ]
* 差分を確認してからコミット
> git commit -v

## コミット(リモート)
* リモートリポジトリにコミット
> git push -u origin master
